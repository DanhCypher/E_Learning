Dự án Take Exam

Dự án Take Exam là một ứng dụng web cho phép người dùng tham gia các kỳ thi trực tuyến. Ứng dụng này được xây dựng bằng công nghệ HTML, CSS, JavaScript và sử dụng framework React để tạo ra giao diện người dùng.

Tính năng

Người dùng có thể tham gia các kỳ thi trực tuyến
Ứng dụng hiển thị các câu hỏi và đáp án cho người dùng
Người dùng có thể chọn đáp án và nộp bài
Ứng dụng hiển thị kết quả thi cho người dùng
Cài đặt

Clone dự án từ GitHub: git clone https://github.com/DanhCypher/E_Learning.git
Cài đặt các gói cần thiết: npm install
Khởi động ứng dụng: npm start
Cấu trúc dự án

public: chứa các file HTML, CSS, JavaScript cho ứng dụng
controller: chứa các đoạn code logic backend cho các tính năng đăng nhâp, đăng ký, các chức năng: thêm, xóa, sửa, đọc cho đề thi và câu hỏi
route: các đường dẫn liên quan
model: cơ sở dữ liệu
